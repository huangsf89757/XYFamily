# 07-超级管理员接口设计

> P3 接口契约之七。定义 SuperAdmin 专属接口：初始化、系统配置查看/修改、强制降级、全局审计日志查询。SuperAdmin 不依赖任何 membership（[P2 RBAC §二](../../02-核心域/02-RBAC权限引擎方案.md)），权限来自 `super_admin` 角色，错误码段 `80xxx`（系统）。

## 一、鉴权范围

- 全部接口需 `super_admin` 角色（全局，无 scope Header）。
- 高危操作（强制降级、保留期缩短）需二次确认（FR-ADMIN-007）。
- 错误码段 `80xxx`；参数 `100001`、频控 `429`。

---

## 二、超级管理员初始化（鸡生蛋）

`POST /api/v1/admin/init`　**公开（仅数据库无 SuperAdmin 时）**

请求（生产建议走 CLI，[ADR-001](../../01-基座/02-ADR架构决策记录.md)）：`{ "email":"admin@x.com", "password":"Str0ng@Pass12!" }`
响应 `201`：`{ "data": { "account_id", "require_password_change": true } }`（首次登录强制改密）
错误：`800002`（已存在 SuperAdmin，幂等拒绝）。

---

## 三、系统配置查看

`GET /api/v1/admin/config`　**admin.config.read**
响应 `200`：`{ "data": { "token.access_ttl":1800, "token.refresh_ttl":604800, "login.rate_limit.threshold":5, "verification_code.ttl":300, "audit.retention_days":365, "account.deactivate_grace_days":30, "login.auto_register":false } }`

---

## 四、系统配置修改

`PUT /api/v1/admin/config`　**admin.config.update**（高风险，二次确认）
请求：`{ "config_key":"token.access_ttl", "config_value":"3600", "confirm": true }`
响应 `200`：`{ "data": { "config_key", "config_value", "updated_at" } }`（即时生效，记录审计 `admin.config.update`）
错误：`800003`（配置项不存在/越界/未二次确认）。

---

## 五、强制降级

`POST /api/v1/admin/force-downgrade`　**admin.force_downgrade**（高风险，二次确认）
请求：`{ "scope_type":"organization", "scope_id":"uuid", "account_id":"uuid", "confirm": true }`
响应 `200`：`{ "data": { "account_id", "before_role", "after_role":"regular_member" } }`
说明：突破「最后管理员保护」，可清空某 scope 全部管理员（FR-ADMIN-005）。
错误：`800004`（目标不可降级/SuperAdmin 自身）。

---

## 六、全局审计日志查询

`GET /api/v1/admin/audit-logs?org_id=&account_id=&action_type=&result=&start=&end=&page=1&size=20`　**admin.audit.global**
响应 `200`：`{ "data": { "items":[{"id","account_id","action_domain","action_type","target_type","result","created_at"}], "total" } }`
错误：`800005`（查询参数非法）。

---

## 七、审计日志详情

`GET /api/v1/admin/audit-logs/{id}`　**admin.audit.global**
响应 `200`：`{ "data": { "id","account_id","org_id","action_domain","action_type","target_type","target_id","details","ip_address","user_agent","created_at" } }`

---

## 八、错误码段（80xxx 摘要）

| code | 说明 |
|------|------|
| 800001 | 服务器内部错误（全局） |
| 800002 | SuperAdmin 已存在 |
| 800003 | 配置非法/未二次确认 |
| 800004 | 目标不可强制降级 |
| 800005 | 查询参数非法 |
