## 一、鉴权范围

- 组织域查询需 `X-Organization-ID`，仅 `organization_core_admin` 可查本组织（FR-AUDIT-007）。
- 全局查询（`/admin/audit-logs`）仅 SuperAdmin（见 [超管接口](./07-超管接口.md)）。
- 审计日志**仅追加不可篡改**（FR-AUDIT-009），无写入/删除接口对外开放。
- 错误码段 `70xxx`。

---

## 二、查询审计日志（本组织）

`GET /api/v1/audit-logs?account_id=&action_type=&result=&start=&end=&page=1&size=20`　**audit.query / audit.operation.read / audit.login.read**

请求需 `X-Organization-ID`。
响应 `200`：
```json
{ "data": { "items": [
  { "id":"uuid", "account_id":"uuid", "action_domain":"org",
    "action_type":"org.role.assign", "target_type":"org_member",
    "result":"success", "created_at":"2026-..." } ], "total": 120 } }
```
错误：`700001`（无查询权限）/ `700002`（查询参数非法）。

> 筛选维度：时间范围、账号、操作类型、登录结果；分页按 `created_at DESC`。

---

## 三、审计日志详情

`GET /api/v1/audit-logs/{id}`　**audit.query**

响应 `200`：
```json
{ "data": {
  "id":"uuid", "account_id":"uuid", "org_id":"uuid",
  "action_domain":"account", "action_type":"account.password.update",
  "target_type":"account", "target_id":"uuid",
  "details": {"changed":true},
  "ip_address":"1.2.3.4", "user_agent":"...", "created_at":"2026-..." } }
```
> `details` 为前后快照 JSONB；密码/Token 不落明文（FR-AUDIT-010）。PII 展示脱敏。

---

## 四、错误码段（70xxx 摘要）

| code | 说明 |
|------|------|
| 700001 | 无审计查询权限 |
| 700002 | 查询参数非法 |
| 700003 | 日志不存在/越权访问 |
