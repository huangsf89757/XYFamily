# 小组接口

> P3 接口契约之五。定义小组管理模块 API（信息 / 删除 / 成员 / 角色）。鉴权范围需携带 `X-Organization-ID` + `X-Team-ID` + `X-Group-ID`，三级 scope 必须自洽（[P2 多租户隔离 §四](../../02-核心域/01-多租户隔离方案.md)）。

## 一、鉴权范围

- 需 `X-Organization-ID` + `X-Team-ID` + `X-Group-ID`；三级 scope 必须由下至上自洽，否则 `400`。
- 团队核心管理员经继承自动拥有该团队下所有小组核心权限（[P2 RBAC §四](../../02-核心域/02-RBAC权限引擎方案.md)）。
- 错误码段 `50xxx`。

---

## 二、获取小组信息

`GET /api/v1/groups/{group_id}`　**group.read**
响应 `200`：`{ "data": { "group_id", "team_id", "org_id", "name", "description" } }`

---

## 三、更新小组信息

`PUT /api/v1/groups/{group_id}`　**group.update**（小组核心 / 团队核心）
请求：`{ "name": "...", "description": "..." }`
错误：`500004`（无修改权限）。

---

## 四、删除小组

`DELETE /api/v1/groups/{group_id}`　**group.delete**（小组核心 / 团队核心）
响应 `200`：`{ "data": null }`（软删除，保留历史，FR-GROUP-003）

---

## 五、邀请成员

`POST /api/v1/groups/{group_id}/members/invitations`　**group.member.invite**（小组核心/普通 / 团队核心）
请求：`{ "scope_type": "group", "scope_id": "{group_id}", "invitee": "...", "role": "regular_member" }`
响应 `201`：`{ "data": { "invitation_id", "status": "pending" } }`

---

## 六、移除成员

`DELETE /api/v1/groups/{group_id}/members/{account_id}`　**group.member.remove**

---

## 七、获取成员列表

`GET /api/v1/groups/{group_id}/members?page=1&size=20`　**group.member.list**

---

## 八、分配角色

`PUT /api/v1/groups/{group_id}/members/{account_id}/role`　**group.role.assign**（小组核心 / 团队核心）
请求：`{ "role": "group_core_admin" }`
错误：`500007`（角色非法）。

---

## 九、降级核心管理员

`POST /api/v1/groups/{group_id}/members/{account_id}/downgrade`　**group.role.downgrade**（小组核心 / 团队核心）
错误：`500008`（最后管理员不可降级）。

---

## 十、错误码段（50xxx 摘要）

| code | 说明 |
|------|------|
| 500001 | 小组参数错误 |
| 500003 | 小组不存在/无访问权限 |
| 500004 | 无修改权限 |
| 500005 | 邀请失败 |
| 500006 | 最后管理员不可移除 |
| 500007 | 角色非法 |
| 500008 | 最后管理员不可降级 |
