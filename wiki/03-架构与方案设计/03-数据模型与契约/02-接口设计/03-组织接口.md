## 一、鉴权范围

- 组织域接口需携带 `X-Organization-ID` Header（单组织用户可省略，由 Token `org_ids` 推断；多组织用户必填，否则 `400`）。
- Path 中的 `{organization_id}` 必须与 Header 自洽，否则 `400`。
- 错误码段 `30xxx`。

---

## 二、创建组织

`POST /api/v1/organizations`　**org.create**（仅 SuperAdmin，见 [权限矩阵](../../02-需求与产品设计/01-产品PRD/01-多租户底座/06-权限管理模块/权限管理模块-V1.0.0.md)）

请求：`{ "name": "XX家庭", "description": "...", "initial_admin": "13800000000" }`（initial_admin 走统一邀请流，ADR-014）
响应 `201`：`{ "data": { "org_id", "name", "owner_id" } }`
错误：`300001`（名称重复/参数）/ `300002`（无权限创建组织）。

> 首版组织创建由 SuperAdmin 入口负责（ADR-001/014「鸡生蛋」），普通用户经邀请加入。

---

## 三、获取组织信息

`GET /api/v1/organizations/{organization_id}`　**org.read**

响应 `200`：`{ "data": { "org_id", "name", "description", "owner_id", "created_at" } }`
错误：`300003`（组织不存在/无权限）。

---

## 四、更新组织信息

`PUT /api/v1/organizations/{organization_id}`　**org.update**（组织核心）

请求：`{ "name": "...", "description": "..." }`
响应 `200`：`{ "data": { "org_id", "name", "description" } }`
错误：`300004`（无修改权限）。

---

## 五、邀请成员

`POST /api/v1/organizations/{organization_id}/members/invitations`　**org.member.invite**（组织核心/普通）

请求：`{ "scope_type": "organization", "scope_id": "{org_id}", "invitee": "13800000000", "role": "regular_member" }`（role 必填，ADR-009）
响应 `201`：`{ "data": { "invitation_id", "status": "pending" } }`
错误：`300005`（邀请失败/已存在成员）。

---

## 六、移除成员

`DELETE /api/v1/organizations/{organization_id}/members/{account_id}`　**org.member.remove**

响应 `200`：`{ "data": null }`
错误：`300006`（最后管理员不可移除）。

---

## 七、获取成员列表

`GET /api/v1/organizations/{organization_id}/members?page=1&size=20`　**org.member.list**

响应 `200`：`{ "data": { "items": [{"account_id","role","nickname","joined_at"}], "total" } }`

---

## 八、分配角色

`PUT /api/v1/organizations/{organization_id}/members/{account_id}/role`　**org.role.assign**（组织核心）

请求：`{ "role": "organization_core_admin" }`
响应 `200`：`{ "data": { "account_id", "role" } }`
错误：`300007`（角色非法）。

---

## 九、降级核心管理员

`POST /api/v1/organizations/{organization_id}/members/{account_id}/downgrade`　**org.role.downgrade**（组织核心）

响应 `200`：`{ "data": { "account_id", "role": "regular_member" } }`
错误：`300008`（最后管理员不可降级）。

---

## 十、创建团队

`POST /api/v1/organizations/{organization_id}/teams`　**org.team.create**（组织核心）

请求：`{ "name": "...", "description": "...", "initial_admin": "..." }`
响应 `201`：`{ "data": { "team_id", "org_id", "name" } }`
错误：`300009`（无创建权限）。

---

## 十一、错误码段（30xxx 摘要）

| code | 说明 |
|------|------|
| 300001 | 组织参数错误 |
| 300002 | 无创建组织权限 |
| 300003 | 组织不存在/无访问权限 |
| 300004 | 无修改权限 |
| 300005 | 邀请失败（已成员/非法角色） |
| 300006 | 最后管理员不可移除 |
| 300007 | 角色非法 |
| 300008 | 最后管理员不可降级 |
| 300009 | 无创建团队权限 |
