## 一、通用约定

- 均需 `Authorization: Bearer <Access Token>`，**无 scope Header**（个人域）。
- 错误码段 `20xxx`（账号）：`200001` 密码强度、`200005` 新密码相同、`200007` 宽限期中、`200010` 已注销不可操作等。

---

## 二、获取个人信息

`GET /api/v1/account/profile`　**account.profile.read**

响应 `200`：
```json
{ "data": {
  "account_id": "uuid", "username": "user_xxx",
  "phone": "138****0000", "email": "a***@x.com",   // PII 脱敏
  "nickname": "小明", "avatar": "https://...", "status": "active" } }
```

---

## 三、更新个人信息

`PUT /api/v1/account/profile`　**account.profile.update**

请求：`{ "nickname": "新昵称", "avatar": "https://..." }`（仅可编辑字段）
响应 `200`：`{ "data": { "account_id", "nickname", "avatar" } }`
错误：`100001`（参数错误）。

---

## 四、修改密码（需登录）

`PUT /api/v1/account/password`　**account.password.update**

请求：`{ "old_password": "...", "new_password": "New@Pass34" }`
响应 `200`：`{ "data": null }`（除当前会话外其他会话失效，FR-ACCT-003）
错误：`200001`（强度不足）/ `200011`（旧密码错误）/ `200005`（新密码相同）。

---

## 五、账号注销（进入宽限期）

`POST /api/v1/account/deactivate`　**account.deactivate**

请求：`{}`
响应 `200`：`{ "data": { "status": "deactivating", "deactivated_at": "<ts+30d>" } }`
说明：进入 30 天宽限期，期间不可执行业务操作（FR-ACCT-004，ADR-010）。

---

## 六、账号恢复（宽限期内）

`POST /api/v1/account/undeactivate`　**account.undeactivate**

请求：`{ "type": "phone|email", "target": "...", "code": "123456" }`
响应 `200`：`{ "data": { "status": "active" } }`（角色权限自动恢复，FR-ACCT-005）
错误：`110001~110004`（验证码）/ `200007`（已超宽限期不可恢复）。
