## 必含项（每份统一）

- 统一 JSON 信封：`{code, message, detail, data}`（ADR-013）。
- 错误码表：HTTP Status（401/403/400/404/429/500）+ 业务 Code（10xxx~80xxx）。
- 鉴权范围标注：哪些接口需带 `X-Org/Team/Group-ID`、返回 401/403 的时机。
- 限流与防重放要求（NFR-SEC-002、防重放）。
- 版本策略：统一前缀 `/api/v1/`，旧版保留 ≥ 6 月。
