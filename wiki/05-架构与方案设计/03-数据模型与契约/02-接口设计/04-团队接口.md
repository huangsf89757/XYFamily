# 团队接口

> P3 接口契约之四。定义团队管理模块 API（信息 / 成员 / 角色 / 创建小组 / 归档）。与组织接口结构一致，鉴权范围需同时携带 `X-Organization-ID` 与 `X-Team-ID`，父子 scope 须自洽（[多租户隔离方案](../../02-核心域/01-多租户隔离方案.md)）。

## 一、鉴权范围

- 需 `X-Organization-ID` + `X-Team-ID`；`X-Team-ID` 所属组织必须等于 `X-Organization-ID`，否则 `400`。
- 组织核心管理员经层级回溯可操作团队（[RBAC权限引擎方案](../../02-核心域/02-RBAC权限引擎方案.md)）。
- 错误码段 `40xxx`。

---

## 二、获取团队信息

`GET /api/v1/teams/{team_id}`　**team.read**
响应 `200`：`{ "data": { "team_id", "org_id", "name", "description", "archived_at" } }`

---

## 三、更新团队信息

`PUT /api/v1/teams/{team_id}`　**team.update**（团队核心 / 组织核心）
请求：`{ "name": "...", "description": "..." }`
响应 `200`：`{ "data": { "team_id", "name", "description" } }`
错误：`400004`（无修改权限）。

---

## 四、邀请成员

`POST /api/v1/teams/{team_id}/members/invitations`　**team.member.invite**（团队核心/普通 / 组织核心/普通）
请求：`{ "scope_type": "team", "scope_id": "{team_id}", "invitee": "...", "role": "regular_member" }`
响应 `201`：`{ "data": { "invitation_id", "status": "pending" } }`

---

## 五、移除成员

`DELETE /api/v1/teams/{team_id}/members/{account_id}`　**team.member.remove**
错误：`400006`（最后管理员不可移除）。

---

## 六、获取成员列表

`GET /api/v1/teams/{team_id}/members?page=1&size=20`　**team.member.list**
响应 `200`：`{ "data": { "items":[...], "total" } }`

---

## 七、分配角色

`PUT /api/v1/teams/{team_id}/members/{account_id}/role`　**team.role.assign**（团队核心 / 组织核心）
请求：`{ "role": "team_core_admin" }`
错误：`400007`（角色非法）。

---

## 八、降级核心管理员

`POST /api/v1/teams/{team_id}/members/{account_id}/downgrade`　**team.role.downgrade**（团队核心 / 组织核心）
错误：`400008`（最后管理员不可降级）。

---

## 九、创建小组

`POST /api/v1/teams/{team_id}/groups`　**team.group.create**（团队核心 / 组织核心）
请求：`{ "name": "...", "description": "...", "initial_admin": "..." }`
响应 `201`：`{ "data": { "group_id", "team_id", "name" } }`

---

## 十、归档团队

`POST /api/v1/teams/{team_id}/archive`　**team.archive**（团队核心 / 组织核心）
响应 `200`：`{ "data": { "team_id", "archived_at": "<ts>" } }`（软归档，FR-TEAM-009）

---

## 十一、错误码段（40xxx 摘要）

| code | 说明 |
|------|------|
| 400001 | 团队参数错误 |
| 400003 | 团队不存在/无访问权限 |
| 400004 | 无修改权限 |
| 400005 | 邀请失败 |
| 400006 | 最后管理员不可移除 |
| 400007 | 角色非法 |
| 400008 | 最后管理员不可降级 |
| 400009 | 无创建小组权限 |
