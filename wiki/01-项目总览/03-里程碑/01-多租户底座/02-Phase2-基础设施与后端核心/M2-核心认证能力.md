## 四、技术要点

- JWT：Access Token 30min + Refresh Token 7天（NFR-SEC-004），HS256（可升级 RS256，NFR-SEC-005）。
- 验证码：6 位数字，有效期 5 分钟（NFR-SEC-003）。
- 限流：同一 IP 5 分钟 5 次失败锁定 15 分钟（NFR-SEC-002）。
- 所有接口强制 HTTPS（NFR-SEC-008）。

## 五、接口清单（代表）

- `POST /api/v1/auth/register/phone`、`POST /api/v1/auth/register/email`
- `POST /api/v1/auth/login`（支持 password / code 模式）
- `POST /api/v1/auth/refresh`、`POST /api/v1/auth/logout`
- `POST /api/v1/auth/password/reset`、`POST /api/v1/auth/verification-code/send`

## 六、质量门禁与 DOD

- **DOD**：单测覆盖认证主链路 ≥ 80%；登录限流与验证码策略单测通过；接口契约与 OpenAPI 一致；SAST 零高危。
