## 四、技术要点

### 4.1 技术栈

| 类别 | 选型 | 版本 |
|------|------|------|
| 框架 | React | 18+ |
| 语言 | TypeScript | 5.x |
| 构建工具 | Vite | 5.x |
| UI 组件库 | Ant Design | 5.x |
| 图表 | ECharts | 5.x |
| HTTP | axios | latest |
| 路由 | React Router | 6.x |
| 状态管理 | Zustand / Redux Toolkit | - |

### 4.2 API 调用层

```typescript
// 统一拦截器
- 请求拦截: 自动附加 Authorization Header + X-Organization-ID
- 响应拦截: 401 → 自动刷新 Token → 重试 → 失败跳转登录
- 错误处理: 统一 message 提示，错误码映射文案
```

### 4.3 多租户上下文

- 用户选择当前操作的组织 → 存储到全局状态
- 所有受保护 API 请求自动附加 `X-Organization-ID` Header
- 切换组织时刷新权限 + 清除页面缓存

### 4.4 权限路由

- 路由配置声明所需权限点
- 路由守卫校验当前用户角色是否拥有权限
- 无权限 → 重定向到 403 页面
