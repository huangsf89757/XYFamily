## 四、技术要点

### 4.1 微信登录流程

```
前端发起微信授权 → 微信回调 code → 后端用 code 换 access_token + openid
  → 查询 third_party_identities
    ├── 已绑定 → 返回 JWT Token（登录成功）
    └── 未绑定 → 自动创建 account + 绑定 openid → 返回 JWT Token
```

### 4.2 third_party_identities 表设计（预期）

```sql
CREATE TABLE third_party_identities (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  account_id UUID NOT NULL REFERENCES accounts(id),
  provider VARCHAR(20) NOT NULL,        -- wechat / alipay
  open_id VARCHAR(255) NOT NULL,
  union_id VARCHAR(255),                -- 微信 union_id（可选）
  nickname VARCHAR(100),
  avatar_url TEXT,
  created_at TIMESTAMP NOT NULL DEFAULT CURRENT_TIMESTAMP,
  UNIQUE(provider, open_id)
);
```

### 4.3 登录方式扩展

`login_audit_logs.login_method` 新增 `wechat` 枚举值。
