## 四、技术要点

### 4.1 H5 技术栈

| 类别 | 选型 | 版本 |
|------|------|------|
| 框架 | React | 18+ |
| 语言 | TypeScript | 5.x |
| 构建工具 | Vite | 5.x |
| UI 组件库 | Vant | 4.x |
| HTTP | axios | latest |
| 路由 | React Router | 6.x |

### 4.2 小程序技术栈

| 类别 | 选型 | 说明 |
|------|------|------|
| 框架 | 原生 / uni-app | 待定 |
| 语言 | TypeScript | - |
| UI 组件库 | Vant Weapp / uView | - |
| HTTP | wx.request 封装 | - |

### 4.3 Token 管理

| 平台 | Access Token | Refresh Token |
|------|-------------|---------------|
| H5 | localStorage | localStorage |
| 小程序 | wx.setStorageSync | wx.setStorageSync |

### 4.4 统一 API 层

- Token 自动刷新：401 时自动调用 `/auth/refresh`，成功后重试原请求
- 多租户 Header：自动附加 `X-Organization-ID`
- 统一错误码处理：错误码 → 中文文案映射
