## 规范层级关系

```
全局设计规范（设计令牌层）
├── 网页端设计规范（Web 适配层）
├── 桌面端设计规范（Desktop 适配层）
├── 移动端设计规范（Mobile 适配层）
└── 小程序设计规范（Mini Program 适配层）
```

> 各端规范以全局设计规范为基础，补充端特有的布局、交互、组件和平台适配规则。全局令牌（色彩、字体、间距等）在各端保持一致，各端仅在尺度、密度和交互模式上做差异化适配。
