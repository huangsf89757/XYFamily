## 一、创建组织

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| 方法 | `POST` |
| 路径 | `/api/v1/org` |
| 认证 | ✅ 需 Token |
| HTTP Status | 201 |
| 限流 | 每账号每天创建 3 个组织 |

### 1.2 请求体

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `name` | `string` | ✅ | 组织名称（2-100 字符） |
| `description` | `string` | 否 | 组织描述（0-500 字符） |

**请求示例**：

```json
{
  "name": "星辰科技有限公司",
  "description": "一家专注于人工智能领域的创新科技公司"
}
```

### 1.3 响应体

**成功响应**：

```json
{
  "code": 0,
  "message": "success",
  "detail": null,
  "data": {
    "org_id": "123e4567-e89b-12d3-a456-426614174000",
    "name": "星辰科技有限公司",
    "description": "一家专注于人工智能领域的创新科技公司",
    "status": "active",
    "owner_account_id": "123e4567-e89b-12d3-a456-426614174001",
    "created_at": "2026-07-16T09:00:00Z",
    "updated_at": "2026-07-16T09:00:00Z"
  },
  "pagination": null
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `org_id` | `string` (UUID) | 组织 ID |
| `name` | `string` | 组织名称 |
| `description` | `string` | 组织描述 |
| `status` | `string` | 组织状态：`active`/`inactive` |
| `owner_account_id` | `string` (UUID) | 创建者账号 ID（自动赋予 `organization_core_admin` 角色） |
| `created_at` | `string` | 创建时间（ISO 8601 UTC） |
| `updated_at` | `string` | 更新时间（ISO 8601 UTC） |

**错误响应**：

```json
{
  "code": 300002,
  "message": "组织名称已存在",
  "detail": "该名称已被其他组织使用，请更换名称",
  "data": null,
  "pagination": null
}
```

### 1.4 错误码

| HTTP | 业务 Code | 说明 |
|------|----------|------|
| 400 | 100001 | 参数类型不合法 |
| 400 | 100002 | 参数值不合法 |
| 400 | 100003 | 必填字段缺失 |
| 400 | 300002 | 组织名称已存在 |
| 401 | 101001 | 缺少 Token |
| 401 | 101002 | Token 无效或已过期 |
| 429 | 80290 | 创建组织频控 |

## 七、移除组织成员

### 7.1 基本信息

| 项目 | 内容 |
|------|------|
| 方法 | `DELETE` |
| 路径 | `/api/v1/org/:org_id/members/:account_id` |
| 认证 | ✅ 需 Token |
| HTTP Status | 200 |
| 限流 | 无 |

### 7.2 请求体

无请求体。

### 7.3 响应体

**成功响应**：

```json
{
  "code": 0,
  "message": "success",
  "detail": null,
  "data": null,
  "pagination": null
}
```

**业务说明**：
- **最后管理员保护**——组织仅剩一个 `organization_core_admin` 时，不允许移除
- 移除成员会同时移除其在该组织下所有团队和小组的角色

**错误响应**：

```json
{
  "code": 300006,
  "message": "最后管理员约束",
  "detail": "该成员是组织最后一名管理员，不允许移除",
  "data": null,
  "pagination": null
}
```

### 7.4 错误码
