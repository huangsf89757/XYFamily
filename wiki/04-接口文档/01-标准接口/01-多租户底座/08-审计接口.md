## 一、获取组织审计日志

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| 方法 | `GET` |
| 路径 | `/api/v1/org/:org_id/audit-logs` |
| 认证 | ✅ 需 Token |
| HTTP Status | 200 |
| 限流 | 无 |

### 1.2 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `page` | `int` | 否 | 页码，默认 1 |
| `page_size` | `int` | 否 | 每页数量，默认 20，最大 100 |
| `account_id` | `string` (UUID) | 否 | 操作人账号 ID |
| `action` | `string` | 否 | 操作类型 |
| `target_type` | `string` | 否 | 目标类型：`org`/`team`/`group`/`member`/`role` |
| `from` | `string` | 否 | 起始时间（ISO 8601） |
| `to` | `string` | 否 | 结束时间（ISO 8601） |
| `result` | `string` | 否 | 操作结果：`success`/`failed` |

### 1.3 响应体

**成功响应**：

```json
{
  "code": 0,
  "message": "success",
  "detail": null,
  "data": {
    "items": [
      {
        "log_id": "123e4567-e89b-12d3-a456-426614177000",
        "account_id": "123e4567-e89b-12d3-a456-426614174001",
        "nickname": "张三",
        "action": "org_create",
        "action_desc": "创建组织",
        "target_type": "org",
        "target_id": "123e4567-e89b-12d3-a456-426614174000",
        "target_name": "星辰科技有限公司",
        "result": "success",
        "ip": "192.168.1.100",
        "user_agent": "Mozilla/5.0",
        "created_at": "2026-07-16T09:00:00Z"
      },
      {
        "log_id": "123e4567-e89b-12d3-a456-426614177001",
        "account_id": "123e4567-e89b-12d3-a456-426614174001",
        "nickname": "张三",
        "action": "member_add",
        "action_desc": "添加成员",
        "target_type": "member",
        "target_id": "123e4567-e89b-12d3-a456-426614174002",
        "target_name": "李四",
        "result": "success",
        "ip": "192.168.1.100",
        "user_agent": "Mozilla/5.0",
        "created_at": "2026-07-16T10:00:00Z"
      }
    ],
    "total": 2,
    "page": 1,
    "page_size": 20
  },
  "pagination": null
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `log_id` | `string` (UUID) | 日志 ID |
| `account_id` | `string` (UUID) | 操作人账号 ID |
| `nickname` | `string` | 操作人昵称 |
| `action` | `string` | 操作类型标识 |
| `action_desc` | `string` | 操作类型描述 |
| `target_type` | `string` | 操作目标类型 |
| `target_id` | `string` | 操作目标 ID |
| `target_name` | `string` | 操作目标名称 |
| `result` | `string` | 操作结果 |
| `ip` | `string` | 操作 IP |
| `user_agent` | `string` | 用户代理 |
| `created_at` | `string` | 操作时间（ISO 8601 UTC） |

### 1.4 错误码

| HTTP | 业务 Code | 说明 |
|------|----------|------|
| 400 | 100002 | 参数值不合法 |
| 401 | 101001 | 缺少 Token |
| 401 | 101002 | Token 无效或已过期 |
| 403 | 600001 | 权限不足 |
| 403 | 600003 | 不在当前组织 scope 内 |
| 404 | 300001 | 组织不存在 |
