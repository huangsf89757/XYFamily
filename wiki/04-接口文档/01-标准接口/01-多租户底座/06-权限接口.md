## 一、权限自检

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| 方法 | `GET` |
| 路径 | `/api/v1/permissions/check` |
| 认证 | ✅ 需 Token |
| HTTP Status | 200 |
| 限流 | 无 |

### 1.2 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `permission` | `string` | ✅ | 权限点标识 |
| `org_id` | `string` (UUID) | ⚠️ | 组织 ID（组织级权限点必填） |
| `team_id` | `string` (UUID) | ⚠️ | 团队 ID（团队级权限点必填） |
| `group_id` | `string` (UUID) | ⚠️ | 小组 ID（小组级权限点必填） |

### 1.3 响应体

**成功响应**：

```json
{
  "code": 0,
  "message": "success",
  "detail": null,
  "data": {
    "has_permission": true,
    "permission": "organization_member_manage",
    "scope": "org",
    "org_id": "123e4567-e89b-12d3-a456-426614174000"
  },
  "pagination": null
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `has_permission` | `boolean` | 是否有权限 |
| `permission` | `string` | 权限点标识 |
| `scope` | `string` | 权限作用域：`org`/`team`/`group` |
| `org_id` | `string` (UUID) | 组织 ID |
| `team_id` | `string` (UUID)/`null` | 团队 ID（团队级权限点返回） |
| `group_id` | `string` (UUID)/`null` | 小组 ID（小组级权限点返回） |

**错误响应**：

```json
{
  "code": 600002,
  "message": "权限点不存在",
  "detail": "指定的权限点标识不存在，请检查权限点参数",
  "data": null,
  "pagination": null
}
```

### 1.4 错误码

| HTTP | 业务 Code | 说明 |
|------|----------|------|
| 400 | 100003 | 必填字段缺失 |
| 400 | 600002 | 权限点不存在 |
| 401 | 101001 | 缺少 Token |
| 401 | 101002 | Token 无效或已过期 |

## 二、获取当前用户有效权限列表

### 2.1 基本信息

| 项目 | 内容 |
|------|------|
| 方法 | `GET` |
| 路径 | `/api/v1/permissions/effective` |
| 认证 | ✅ 需 Token |
| HTTP Status | 200 |
| 限流 | 无 |

### 2.2 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `org_id` | `string` (UUID) | 否 | 组织 ID，不传则返回所有组织的权限 |

### 2.3 响应体

**成功响应**：

```json
{
  "code": 0,
  "message": "success",
  "detail": null,
  "data": {
    "org_permissions": [
      "organization_member_manage",
      "organization_role_manage",
      "organization_team_manage"
    ],
    "team_permissions": [
      {
        "team_id": "123e4567-e89b-12d3-a456-426614175000",
        "permissions": [
          "team_member_manage",
          "team_role_manage",
          "team_group_manage"
        ]
      }
    ],
    "group_permissions": [
      {
        "group_id": "123e4567-e89b-12d3-a456-426614176000",
        "permissions": [
          "group_member_manage",
          "group_role_manage"
        ]
      }
    ]
  },
  "pagination": null
}
```

| HTTP | 业务 Code | 说明 |
|------|----------|------|
| 401 | 101001 | 缺少 Token |
| 401 | 101002 | Token 无效或已过期 |
