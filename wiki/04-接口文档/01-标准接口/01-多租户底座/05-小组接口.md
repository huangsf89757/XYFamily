## 一、创建小组

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| 方法 | `POST` |
| 路径 | `/api/v1/org/:org_id/teams/:team_id/groups` |
| 认证 | ✅ 需 Token |
| HTTP Status | 201 |
| 限流 | 每团队最多创建 50 个小组 |

### 1.2 请求体

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `name` | `string` | ✅ | 小组名称（2-100 字符） |
| `description` | `string` | 否 | 小组描述（0-500 字符） |

**请求示例**：

```json
{
  "name": "前端组",
  "description": "负责前端开发工作"
}
```

### 1.3 响应体

**成功响应**：

```json
{
  "code": 0,
  "message": "success",
  "detail": null,
  "data": {
    "group_id": "123e4567-e89b-12d3-a456-426614176000",
    "team_id": "123e4567-e89b-12d3-a456-426614175000",
    "org_id": "123e4567-e89b-12d3-a456-426614174000",
    "name": "前端组",
    "description": "负责前端开发工作",
    "status": "active",
    "created_at": "2026-07-16T09:00:00Z",
    "updated_at": "2026-07-16T09:00:00Z"
  },
  "pagination": null
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `group_id` | `string` (UUID) | 小组 ID |
| `team_id` | `string` (UUID) | 所属团队 ID |
| `org_id` | `string` (UUID) | 所属组织 ID |
| `name` | `string` | 小组名称 |
| `description` | `string` | 小组描述 |
| `status` | `string` | 小组状态：`active`/`deleted` |
| `created_at` | `string` | 创建时间（ISO 8601 UTC） |
| `updated_at` | `string` | 更新时间（ISO 8601 UTC） |

**错误响应**：

```json
{
  "code": 500002,
  "message": "小组名称已存在",
  "detail": "该名称已被同团队下其他小组使用，请更换名称",
  "data": null,
  "pagination": null
}
```

### 1.4 错误码

| HTTP | 业务 Code | 说明 |
|------|----------|------|
| 400 | 100001 | 参数类型不合法 |
| 400 | 100002 | 参数值不合法 |
| 400 | 100003 | 必填字段缺失 |
| 400 | 500002 | 小组名称已存在 |
| 400 | 500003 | 小组数量已达上限 |
| 401 | 101001 | 缺少 Token |
| 401 | 101002 | Token 无效或已过期 |
| 403 | 600001 | 权限不足 |
| 403 | 600003 | 不在当前组织 scope 内 |
| 404 | 300001 | 组织不存在 |
| 404 | 400001 | 团队不存在 |
