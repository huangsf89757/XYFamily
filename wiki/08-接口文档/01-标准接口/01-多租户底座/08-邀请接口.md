## 一、创建邀请

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| 方法 | `POST` |
| 路径 | `/api/v1/org/:org_id/invitations` |
| 认证 | ✅ 需 Token |
| HTTP Status | 201 |
| 限流 | 无 |

### 1.2 请求体

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `target` | `string` | ✅ | 被邀请人手机号或邮箱 |
| `target_type` | `string` (枚举) | ✅ | 目标类型：`phone`/`email` |
| `role` | `string` (枚举) | ✅ | 角色：`organization_admin`/`organization_member` |
| `team_id` | `string` (UUID) | 否 | 同时邀请加入的团队 ID |
| `group_id` | `string` (UUID) | 否 | 同时邀请加入的小组 ID |

**请求示例**：

```json
{
  "target": "13800000001",
  "target_type": "phone",
  "role": "organization_member",
  "team_id": "123e4567-e89b-12d3-a456-426614175000"
}
```

### 1.3 响应体

**成功响应**：

```json
{
  "code": 0,
  "message": "success",
  "detail": null,
  "data": {
    "invitation_id": "123e4567-e89b-12d3-a456-426614178000",
    "org_id": "123e4567-e89b-12d3-a456-426614174000",
    "org_name": "星辰科技有限公司",
    "target": "138****0001",
    "role": "organization_member",
    "status": "pending",
    "inviter_account_id": "123e4567-e89b-12d3-a456-426614174001",
    "inviter_nickname": "张三",
    "expires_at": "2026-07-23T09:00:00Z",
    "created_at": "2026-07-16T09:00:00Z"
  },
  "pagination": null
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `invitation_id` | `string` (UUID) | 邀请 ID |
| `org_id` | `string` (UUID) | 组织 ID |
| `org_name` | `string` | 组织名称 |
| `target` | `string` | 被邀请人手机号/邮箱（脱敏） |
| `role` | `string` | 邀请角色 |
| `status` | `string` | 邀请状态：`pending`/`accepted`/`rejected`/`expired`/`cancelled` |
| `inviter_account_id` | `string` (UUID) | 邀请人账号 ID |
| `inviter_nickname` | `string` | 邀请人昵称 |
| `expires_at` | `string` | 过期时间（ISO 8601 UTC），默认 7 天 |
| `created_at` | `string` | 创建时间（ISO 8601 UTC） |

**错误响应**：

```json
{
  "code": 210003,
  "message": "角色不合法",
  "detail": "指定的角色不在当前组织允许范围内",
  "data": null,
  "pagination": null
}
```

### 1.4 错误码

| HTTP | 业务 Code | 说明 |
|------|----------|------|
| 400 | 100001 | 参数类型不合法 |
| 400 | 100003 | 必填字段缺失 |
| 400 | 100004 | 枚举值不合法 |
| 400 | 210003 | 角色不合法 |
| 401 | 101001 | 缺少 Token |
| 401 | 101002 | Token 无效或已过期 |
| 403 | 600001 | 权限不足 |
| 403 | 600003 | 不在当前组织 scope 内 |
| 404 | 300001 | 组织不存在 |

## 二、接受邀请

### 2.1 基本信息

| 项目 | 内容 |
|------|------|
| 方法 | `POST` |
| 路径 | `/api/v1/invitations/:invitation_id/accept` |
| 认证 | ✅ 需 Token |
| HTTP Status | 200 |
| 限流 | 无 |

### 2.2 请求体

无请求体。

### 2.3 响应体

**成功响应**：

```json
{
  "code": 0,
  "message": "success",
  "detail": null,
  "data": {
    "invitation_id": "123e4567-e89b-12d3-a456-426614178000",
    "org_id": "123e4567-e89b-12d3-a456-426614174000",
    "org_name": "星辰科技有限公司",
    "role": "organization_member",
    "status": "accepted",
    "accepted_at": "2026-07-16T10:00:00Z"
  },
  "pagination": null
}
```

**业务说明**：接受邀请后自动加入组织（及指定的团队/小组），并赋予对应角色。

**错误响应**：

```json
{
  "code": 210001,
  "message": "邀请状态不允许操作",
  "detail": "该邀请已过期或已被处理，无法接受",
  "data": null,
  "pagination": null
}
```

### 2.4 错误码

| HTTP | 业务 Code | 说明 |
|------|----------|------|
| 400 | 100002 | 参数值不合法 |
| 400 | 210001 | 邀请状态不允许操作 |
| 400 | 210002 | account_id 与邀请不匹配 |
| 401 | 101001 | 缺少 Token |
| 401 | 101002 | Token 无效或已过期 |
| 404 | 210404 | 邀请不存在或已过期 |

## 四、获取邀请列表

### 4.1 基本信息

| 项目 | 内容 |
|------|------|
| 方法 | `GET` |
| 路径 | `/api/v1/org/:org_id/invitations` |
| 认证 | ✅ 需 Token |
| HTTP Status | 200 |
| 限流 | 无 |

### 4.2 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `page` | `int` | 否 | 页码，默认 1 |
| `page_size` | `int` | 否 | 每页数量，默认 20，最大 100 |
| `status` | `string` | 否 | 状态筛选：`pending`/`accepted`/`rejected`/`expired`/`cancelled` |

### 4.3 响应体

**成功响应**：

```json
{
  "code": 0,
  "message": "success",
  "detail": null,
  "data": {
    "items": [
      {
        "invitation_id": "123e4567-e89b-12d3-a456-426614178000",
        "target": "138****0001",
        "role": "organization_member",
        "status": "pending",
        "inviter_nickname": "张三",
        "expires_at": "2026-07-23T09:00:00Z",
        "created_at": "2026-07-16T09:00:00Z"
      }
    ],
    "total": 1,
    "page": 1,
    "page_size": 20
```
