## 一、创建团队

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| 方法 | `POST` |
| 路径 | `/api/v1/org/:org_id/teams` |
| 认证 | ✅ 需 Token |
| HTTP Status | 201 |
| 限流 | 每组织最多创建 100 个团队 |

### 1.2 请求体

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `name` | `string` | ✅ | 团队名称（2-100 字符） |
| `description` | `string` | 否 | 团队描述（0-500 字符） |

**请求示例**：

```json
{
  "name": "研发一部",
  "description": "负责核心产品研发"
}
```

### 1.3 响应体

**成功响应**：

```json
{
  "code": 0,
  "message": "success",
  "detail": null,
  "data": {
    "team_id": "123e4567-e89b-12d3-a456-426614175000",
    "org_id": "123e4567-e89b-12d3-a456-426614174000",
    "name": "研发一部",
    "description": "负责核心产品研发",
    "status": "active",
    "created_at": "2026-07-16T09:00:00Z",
    "updated_at": "2026-07-16T09:00:00Z"
  },
  "pagination": null
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `team_id` | `string` (UUID) | 团队 ID |
| `org_id` | `string` (UUID) | 所属组织 ID |
| `name` | `string` | 团队名称 |
| `description` | `string` | 团队描述 |
| `status` | `string` | 团队状态：`active`/`archived` |
| `created_at` | `string` | 创建时间（ISO 8601 UTC） |
| `updated_at` | `string` | 更新时间（ISO 8601 UTC） |

**错误响应**：

```json
{
  "code": 400002,
  "message": "团队名称已存在",
  "detail": "该名称已被同组织下其他团队使用，请更换名称",
  "data": null,
  "pagination": null
}
```

### 1.4 错误码

| HTTP | 业务 Code | 说明 |
|------|----------|------|
| 400 | 100001 | 参数类型不合法 |
| 400 | 100002 | 参数值不合法 |
| 400 | 100003 | 必填字段缺失 |
| 400 | 400002 | 团队名称已存在 |
| 400 | 400003 | 团队数量已达上限 |
| 401 | 101001 | 缺少 Token |
| 401 | 101002 | Token 无效或已过期 |
| 403 | 600001 | 权限不足 |
| 403 | 600003 | 不在当前组织 scope 内 |
| 404 | 300001 | 组织不存在 |

## 七、移除团队成员

### 7.1 基本信息

| 项目 | 内容 |
|------|------|
| 方法 | `DELETE` |
| 路径 | `/api/v1/org/:org_id/teams/:team_id/members/:account_id` |
| 认证 | ✅ 需 Token |
| HTTP Status | 200 |
| 限流 | 无 |

### 7.2 请求体

无请求体。

### 7.3 响应体

**成功响应**：

```json
{
  "code": 0,
  "message": "success",
  "detail": null,
  "data": null,
  "pagination": null
}
```

**业务说明**：
- **最后管理员保护**——团队仅剩一个 `team_core_admin` 时，不允许移除
- 移除成员会同时移除其在该团队下所有小组的角色

**错误响应**：

```json
{
  "code": 400006,
  "message": "最后管理员约束",
  "detail": "该成员是团队最后一名管理员，不允许移除",
  "data": null,
  "pagination": null
}
```

### 7.4 错误码

| HTTP | 业务 Code | 说明 |
|------|----------|------|
