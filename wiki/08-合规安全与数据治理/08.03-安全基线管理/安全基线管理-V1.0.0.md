## 一、代码安全基线

### 1.1 输入验证

| 要求 | 说明 |
|------|------|
| 所有外部输入必须校验 | HTTP Body、Query、Header、Path 参数 |
| 长度限制 | 字符串字段设置最大长度 |
| 格式校验 | 手机号、邮箱、UUID 等格式校验 |
| 白名单 | 枚举值（如 type、status）使用白名单校验 |

### 1.2 错误处理

| 要求 | 说明 |
|------|------|
| 统一错误响应 | `{code, message, detail, data}` 格式 |
| 不泄露敏感信息 | 错误信息不含 SQL、路径、堆栈 |
| 服务端详细日志 | 安全/认证错误记录完整上下文 |
| 客户端友好提示 | 密码错误统一提示"账号或密码错误"，不区分账号不存在 |

### 1.3 安全中间件

| 中间件 | 检查项 |
|--------|--------|
| RateLimiter | 登录限流、验证码发送限流 |
| JWTValidator | Token 签名 + 有效期校验 |
| TokenBlacklist | 登出/改密后 Token 立即失效 |
| PermissionChecker | 权限点校验（含缓存） |
| MembershipValidator | Scope 校验（含层级继承） |

## 二、运行时安全基线

### 2.1 认证与授权

| 要求 | 说明 |
|------|------|
| 所有受保护接口必须认证 | JWT Access Token |
| 所有受保护接口必须校验权限 | PermissionChecker |
| SuperAdmin 单独管理 | 不隶属于任何 org/team/group |
| 最后管理员保护 | 不可降级/移除/注销 |

### 2.2 密码策略

| 要求 | 说明 |
|------|------|
| bcrypt cost=12 | 存储密码哈希 |
| 密码 ≥12 位 | 含大小写+数字+特殊字符 |
| 密码历史 | 新密码不得与旧密码相同 |
| 强制修改 | SuperAdmin 首次登录必须修改密码 |

### 2.3 Session 管理

| 要求 | 说明 |
|------|------|
| Access Token | 30 分钟，无状态 JWT |
| Refresh Token | 7 天，有状态存储 + Rotation |
| Token 黑名单 | Redis 黑名单，TTL = 剩余有效期 |
| 并发限制 | 同一账号并发登录数限制（预留） |

## 三、依赖安全基线

### 3.1 依赖管理

| 要求 | 说明 |
|------|------|
| 锁定版本 | go.mod + go.sum 锁定依赖版本 |
| 定期审计 | CI 集成 `govulncheck` |
| 依赖更新 | 每季度更新依赖，检查安全漏洞 |
| 供应商评估 | 新引入依赖需经过安全审查 |

### 3.2 漏洞管理

| 等级 | 响应时间 | 处理方式 |
|------|---------|---------|
| 高危 | 24 小时内 | 立即修复或降级 |
| 中危 | 1 周内 | 排期修复 |
| 低危 | 下一版本 | 计划修复 |

## 四、部署安全基线

### 4.1 网络

| 要求 | 说明 |
|------|------|
| HTTPS | TLS 1.2+，禁止 HTTP |
| 内网隔离 | 数据库、Redis 仅限内网访问 |
| 最小端口 | 仅开放必要端口（80/443） |

### 4.2 密钥管理

| 要求 | 说明 |
|------|------|
| 环境变量注入 | 数据库密码、JWT Secret 通过环境变量 |
| 禁止硬编码 | 密钥不进代码仓库 |
| 密钥轮换 | JWT Secret 每年轮换 |
| Secret 管理 | 生产环境使用 Vault / KMS |

### 4.3 容器安全

| 要求 | 说明 |
|------|------|
| 最小镜像 | 使用 alpine 基础镜像 |
| 非 root 运行 | 容器内进程不以 root 运行 |
| 健康检查 | `/healthz` 探针 |
| 资源限制 | CPU/Memory limits |

## 五、安全检查清单（每次部署前）

### 5.1 代码审查

- [ ] 无硬编码密钥/密码
- [ ] 所有 SQL 参数化查询
- [ ] 所有外部输入已校验
- [ ] 错误信息不泄露敏感数据
- [ ] 日志不记录密码/Token

### 5.2 依赖审查

- [ ] `govulncheck ./...` 零高危漏洞
- [ ] `golangci-lint run` 零错误
- [ ] go.sum 未发生预期外变更

### 5.3 部署审查

- [ ] 数据库密码已轮换
- [ ] JWT Secret 已更新
- [ ] TLS 证书有效
- [ ] 监控告警已配置
