# 邀请接口

> P3 接口契约之一（草稿）。邀请管理模块的对外 API，覆盖创建/接受/拒绝/取消/查询/待处理六类接口。字段来自 [租户域](../01-数据库设计/02-租户域.md)，鉴权范围遵循 [多租户隔离方案](../../02-核心域/01-多租户隔离方案.md) 的 scope 约定。**正式契约以 [邀请接口](../../../09-接口文档/01-标准接口/01-多租户底座/08-邀请接口.md) 为唯一来源。**

## 一、通用约定

- 统一信封：`{code, message, detail, data}`（ADR-013）。
- 版本前缀：`/api/v1/`。
- 鉴权范围：创建/取消/列表需 `X-Org-ID` 且具备对应 scope 的管理权限；接受/拒绝/待处理使用登录态 `X-Account-ID`。
- 错误码：HTTP 401/403/400/404/409/410/429 + 业务码 21xxx（见 §错误码）。
- 安全：邀请链接/码带签名与有效期，防重放（NFR-SEC-002）。

---

## 二、接口清单

| # | 接口 | Method & Path | 说明 |
|---|------|---------------|------|
| 1 | 创建邀请 | POST `/api/v1/org/:org_id/invitations` | 管理员创建邀请 |
| 2 | 接受邀请 | POST `/api/v1/invitations/:invitation_id/accept` | 被邀请人接受 |
| 3 | 拒绝邀请 | POST `/api/v1/invitations/:invitation_id/reject` | 被邀请人拒绝 |
| 4 | 取消邀请 | DELETE `/api/v1/org/:org_id/invitations/:invitation_id` | 取消未接受邀请 |
| 5 | 查询邀请列表 | GET `/api/v1/org/:org_id/invitations` | 按条件查询 |
| 6 | 查询待处理邀请 | GET `/api/v1/invitations/pending` | 当前账号待处理 |

---

## 2.1 创建邀请

- **Method & Path**：`POST /api/v1/org/:org_id/invitations`
- **Scope**：组织管理员（可下钻至 team/group）
- **Request Body**

```json
{
  "scope_type": "organization | team | group",
  "scope_id": "uuid",
  "role": "regular_member",
  "channel": "email | link | code",
  "invitee_email": "user@example.com",
  "expires_in_days": 7
}
```

- **Response `201`**

```json
{
  "invitation_id": "uuid",
  "scope_type": "organization",
  "scope_id": "uuid",
  "role": "regular_member",
  "status": "pending",
  "channel": "email",
  "invite_link": "https://.../i/xxxx",
  "invite_code": "ABC123",
  "expires_at": "2026-07-26T00:00:00Z",
  "created_at": "2026-07-19T00:00:00Z"
}
```

---

## 2.2 接受邀请

- **Method & Path**：`POST /api/v1/invitations/:invitation_id/accept`
- **Scope**：登录态（`X-Account-ID`）；被邀请人本人
- **Request Body**：`{}`
- **Response `200`**

```json
{
  "membership_id": "uuid",
  "scope_type": "organization",
  "scope_id": "uuid",
  "role": "regular_member",
  "joined_at": "2026-07-19T00:00:00Z"
}
```

---

## 2.3 拒绝邀请

- **Method & Path**：`POST /api/v1/invitations/:invitation_id/reject`
- **Scope**：登录态；被邀请人本人
- **Request Body**：`{}`
- **Response `200`**：`{ "status": "rejected" }`

---

## 2.4 取消邀请

- **Method & Path**：`DELETE /api/v1/org/:org_id/invitations/:invitation_id`
- **Scope**：组织/团队/小组管理员
- **Response `204`**：无内容

---

## 2.5 查询邀请列表

- **Method & Path**：`GET /api/v1/org/:org_id/invitations`
- **Scope**：组织管理员
- **Query**：`scope_type`、`status`、`invitee_email`、`page`、`size`
- **Response `200`**

```json
{
  "items": [
    {
      "invitation_id": "uuid",
      "scope_type": "organization",
      "scope_id": "uuid",
      "role": "regular_member",
      "status": "pending",
      "expires_at": "2026-07-26T00:00:00Z"
    }
  ],
  "total": 1
}
```

---

## 2.6 查询待处理邀请

- **Method & Path**：`GET /api/v1/invitations/pending`
- **Scope**：登录态；返回 `target_account = 当前账号` 的 pending 邀请
- **Response `200`**

```json
{
  "items": [
    {
      "invitation_id": "uuid",
      "scope_type": "organization",
      "scope_id": "uuid",
      "role": "regular_member",
      "expires_at": "2026-07-26T00:00:00Z"
    }
  ]
}
```

## 三、错误码（21xxxx）

| 业务码 | 含义 | HTTP |
|--------|------|------|
| 21001 | 邀请不存在 | 404 |
| 21002 | 邀请已失效/过期 | 410 |
| 21003 | 邀请已被接受/已处理 | 409 |
| 21004 | 邀请码/签名校验失败 | 400 |
| 21005 | 无邀请权限（scope 越权） | 403 |
| 21006 | 邀请数量超限 | 429 |
| 21007 | 被邀请人已在该 scope | 409 |
