## 1. 文档定位

本文件是插件管理 Stage 的测试用例组织规范与索引入口，承接 [Stage 1 测试用例库](../01-多租户底座/03-测试用例库.md) 的通用编写标准，聚焦插件管理功能模块的用例集。

## 2. 用例组织维度（插件管理视角）

- **按模块**：对齐插件管理功能（插件接入框架、插件市场、插件沙箱隔离、插件鉴权与权限）。
- **按类型**：功能用例、接口契约用例、异常 / 边界用例、插件沙箱隔离用例（见 [专项测试](05-专项测试.md) §2）。
- **按端**：四端 + iOS / 鸿蒙 的插件相关差异化用例。

## 3. 用例编写规范

同 Stage 1 通用规范（ID 前缀建议 `PLUG-*`），字段含用例 ID / 标题 / 前置条件（含 X-Organization-ID）/ 步骤 / 预期结果 / 优先级 / 类型。

## 4. 模块用例索引（插件管理）

> 以下为规划中的插件管理用例集，随插件管理里程碑推进逐步补充：

- 插件接入框架用例集（接入流程 / 生命周期 / 版本管理）— 待完善
- 插件市场用例集（上架 / 审核 / 搜索 / 安装 / 卸载）— 待完善
- 插件沙箱隔离用例集（沙箱越权 / 资源隔离 / 钩子安全）— 待完善
- 插件鉴权与权限用例集（插件调用权限 / 租户上下文透传）— 待完善
- 多租户隔离用例集（见 [专项测试](05-专项测试.md) §2）
