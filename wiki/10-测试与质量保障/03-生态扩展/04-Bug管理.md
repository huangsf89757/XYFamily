## 1. 文档定位

本文件是「测试与质量保障」板块在 **Stage 3 生态扩展** 下的 Bug 管理分册。通用缺陷流程见 [Stage 1 Bug 管理](../01-多租户底座/04-Bug管理.md)，本文档只补充**生态类缺陷的严重级别映射**。

## 2. 本 Stage 差异点（生态类缺陷）

| 级别 | 生态扩展示例 |
|------|--------------|
| 致命（Blocker） | 跨组织协作越权导致数据泄露；开放平台越权调用核心接口 |
| 严重（Critical） | 商业化计费错误 / 资损；开放 API 越权访问 |
| 一般（Major） | 协作冲突 / 同步异常、订阅状态不一致 |
| 次要（Minor） | 生态 UI / 文案 / 配置项问题 |

> 涉及跨组织协作越权 / 数据泄露、商业化资损的缺陷一律按"致命"处理。
