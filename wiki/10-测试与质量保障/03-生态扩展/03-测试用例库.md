## 1. 文档定位

本文件是生态扩展 Stage 的测试用例组织规范与索引入口，承接 [Stage 1 测试用例库](../01-多租户底座/03-测试用例库.md) 的通用编写标准，聚焦生态扩展功能模块的用例集。

## 2. 用例组织维度（生态扩展视角）

- **按模块**：对齐生态扩展功能（开放平台、商业化计费、跨组织协作）。
- **按类型**：功能用例、接口契约用例、异常 / 边界用例、跨组织协作隔离用例（见 [专项测试](05-专项测试.md) §2）。
- **按端**：四端 + iOS / 鸿蒙 的生态相关差异化用例。

## 3. 用例编写规范

同 Stage 1 通用规范（ID 前缀建议 `ECO-*`），字段含用例 ID / 标题 / 前置条件（含 X-Organization-ID）/ 步骤 / 预期结果 / 优先级 / 类型。

## 4. 模块用例索引（生态扩展）

> 以下为规划中的生态扩展用例集，随生态扩展里程碑推进逐步补充：

- 开放平台用例集（开放 API / OAuth / 第三方接入）— 待完善
- 商业化用例集（计费 / 订阅 / 支付 / 分账）— 待完善
- 跨组织协作用例集（协作空间 / 跨租户共享 / 邀请）— 待完善
- 生态权限与合规用例集（开放平台权限 / 数据合规）— 待完善
- 多租户隔离用例集（见 [专项测试](05-专项测试.md) §2）
