## 1. 文档定位

本文件是「测试与质量保障」板块在 **Stage 2 插件管理** 下的 Bug 管理分册。通用缺陷流程见 [Stage 1 Bug 管理](../01-多租户底座/04-Bug管理.md)，本文档只补充**插件类缺陷的严重级别映射**。

## 2. 本 Stage 差异点（插件类缺陷）

| 级别 | 插件管理示例 |
|------|--------------|
| 致命（Blocker） | 插件沙箱隔离失效导致跨租户数据泄露；插件越权调用核心接口 |
| 严重（Critical） | 插件市场恶意注入 / 审核绕过；插件导致宿主崩溃 |
| 一般（Major） | 插件版本升级失败、钩子执行异常 |
| 次要（Minor） | 插件 UI / 文案 / 配置项问题 |

> 涉及插件沙箱隔离 / 跨租户数据泄露的缺陷一律按"致命"处理。
