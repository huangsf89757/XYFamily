## 1. 文档定位

本文件定义缺陷从发现到关闭的全生命周期管理规范，缺陷数据与趋势作为门禁评审（G2 / G3）的质量输入之一。

## 2. 缺陷生命周期

```
新建(New) → 确认(Confirmed) → 分配(Assigned) → 修复中(In Progress)
   → 待验证(Resolved) → 已关闭(Closed)
                     ↘ 重新打开(Reopened) → 修复中
   → 挂起(Deferred) / 拒绝(Rejected)
```

## 3. 严重程度定义

| 级别 | 定义 | 示例 |
|------|------|------|
| 致命（Blocker） | 系统崩溃 / 核心流程不可用 / 跨组织数据泄露 | 越权读取其他组织数据（RISK-003） |
| 严重（Critical） | 主要功能失效、无合理绕过 | 登录失败、权限校验失效 |
| 一般（Major） | 功能缺陷但有绕过方案 | 部分列表分页异常 |
| 次要（Minor） | 体验 / 文案 / 边界小问题 | UI 对齐、提示文案 |

> 涉及多租户隔离的缺陷一律按"致命"处理，对应 RISK-003。
