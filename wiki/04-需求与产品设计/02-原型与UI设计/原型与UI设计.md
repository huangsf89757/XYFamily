### 覆盖范围摘要

**网页端（01）** — Web 管理后台完整视图：认证、账号、组织、团队、小组、权限矩阵、超级管理员、审计日志等 9 大模块，响应式布局、明暗双主题。

**桌面端（02）** — 桌面端增强能力 + 复用 Web 业务视图：原生窗口外壳（macOS 交通灯 / Windows 标题栏，可切换）、可折叠侧边栏、`Cmd/Ctrl+K` 命令面板、原生右键菜单、离线 / 更新横幅、7 个业务视图（Dashboard/Org/Team/Group/Permissions/Audit/SuperAdmin）、登录浮层与明暗双主题。

**移动端（03）** — 轻量客户端（iPhone 390×844）：底部 4 Tab（首页 / 组织 / 消息 / 我的）、Inset Grouped 列表、安全区适配、6 个页面（首页 / 组织 / 组织详情 / 消息 / 消息详情 / 我的）、登录浮层、Action Sheet、深色模式开关。

**小程序（04）** — 轻量微信入口（iPhone 375px，rpx 1px=2rpx）：tabBar 4（首页 / 组织 / 消息 / 我的）、微信授权登录（微信绿 `#07C160`）、扫码加入（`wx.scanCode`）、邀请分享（`onShareAppMessage`）、9 个页面、订阅消息 / 分享能力示意、深色模式开关。

> 四端均以「演示切换器 / 前端交互」方式呈现登录、离线、推送、微信生态等系统级能力，无真实后端；管理类重操作按规范收敛至 Web / 桌面端。详见各端 `EXPORT-README.md` 的「已知简化」一节。
