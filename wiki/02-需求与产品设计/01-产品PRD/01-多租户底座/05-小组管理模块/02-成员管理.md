## 一、功能需求

### FR-GROUP-004：邀请成员

| 项目 | 内容 |
|------|------|
| **优先级** | P0 |
| **描述** | 邀请用户加入小组，被邀请人确认后生效 |
| **验收标准** | 邀请发起成功，被邀请人确认后成为小组成员 |

**详细规则：**
- 操作权限：小组核心管理员（`group_core_admin`）/ 小组普通管理员（`group_ordinary_admin`） 或 团队核心管理员（`team_core_admin`）/ 团队普通管理员（`team_ordinary_admin`） 可发起邀请
- 邀请对象：通过手机号、邮箱或账号标识指定被邀请人
- 邀请约束：
  - 同一用户不能重复加入同一小组
  - 被邀请人如果已是该小组成员，拒绝重复邀请
  - 必须指定角色（小组核心管理员（`group_core_admin`）/ 小组普通管理员（`group_ordinary_admin`）/ RegularMember，默认 RegularMember）
  - 被邀请人可在邀请确认页面查看将被分配的角色
- 邀请流程（统一邀请流）：
  1. 管理员发起邀请，系统创建邀请记录，状态为待确认
  2. 系统向被邀请人发送邀请通知（含邀请链接）
  3. 被邀请人点击链接确认，状态变为已接受，创建小组成员记录
  4. 被邀请人也可拒绝，状态变为已拒绝
  5. 邀请有效期 7 天，过期自动失效
- 邀请链接机制：
  - 链接包含邀请标识和一次性校验码
  - 有效期与邀请有效期一致（7 天）
  - 校验码用于防止链接被篡改和滥用
- 未注册用户邀请-注册联动流程：
  1. 被邀请人点击链接，系统校验链接有效性
  2. 如果未注册：
     - 跳转注册页面，预填邀请信息（目标小组、所属团队/组织、将被分配的角色）
     - 用户完成注册（使用同一手机号/邮箱），自动接受邀请并加入目标小组
     - 注册成功后跳转至小组首页
  3. 如果已注册：
     - 跳转邀请确认页面，展示邀请信息（邀请人、目标小组、将被分配的角色）
     - 用户点击接受，加入目标小组
     - 用户点击拒绝，邀请状态变更为已拒绝
- 邀请操作记录审计日志

### FR-GROUP-005：移除成员

| 项目 | 内容 |
|------|------|
| **优先级** | P0 |
| **描述** | 将成员从小组中移除 |
| **验收标准** | 移除后该成员不再属于该小组 |

**详细规则：**
- 操作权限：小组核心管理员（`group_core_admin`）/ 小组普通管理员（`group_ordinary_admin`） 或 团队核心管理员（`team_core_admin`）/ 团队普通管理员（`team_ordinary_admin`） 可移除成员
- 移除约束：
  - 可移除普通成员
  - 不可移除其他小组管理员（需先将其降级为普通成员再移除）
  - 不可移除自己（防止管理员意外退出导致小组无人管理）
- 移除策略为软删除：
  - 小组成员记录标记删除时间，不物理删除
  - 保留成员的历史角色信息，用于审计追溯
- 移除后影响：
  - 该成员的会话中小组上下文立即失效
  - 成员无法继续查看小组信息和成员列表
  - 不影响该成员在其他小组或团队中的成员身份
- 移除操作记录审计日志

### FR-GROUP-006：获取成员列表

| 项目 | 内容 |
|------|------|
| **优先级** | P0 |
| **描述** | 查看小组成员列表，支持分页和筛选 |
| **验收标准** | 返回分页成员列表，支持按角色和用户名筛选 |

**详细规则：**
- 操作权限：小组核心管理员（`group_core_admin`）/ 小组普通管理员（`group_ordinary_admin`） 和 团队核心管理员（`team_core_admin`）/ 团队普通管理员（`team_ordinary_admin`） 可查看；普通成员禁止查看
- 返回信息应包含（每个成员）：账号标识、用户名、显示名、头像、在小组中的角色、加入时间
- 支持分页查询：
  - 默认第 1 页，每页 20 条
  - 最大每页 100 条
- 支持筛选：
  - 按角色筛选（小组核心管理员 / 小组普通管理员 / 普通成员）
  - 按用户名/显示名关键词搜索（模糊匹配）
- 排序规则：默认按加入时间倒序（最新加入在前）

## 二、边界与异常处理

| 场景 | 处理方式 |
|------|----------|
| 邀请已是小组成员的用户 | 提示已是小组成员，拒绝邀请 |
| 邀请时指定非法角色 | 提示角色参数非法，拒绝邀请 |
| 邀请已过期 | 提示邀请不存在或已过期 |
| 移除小组管理员 | 提示需先降级为普通成员，拒绝移除 |
| 移除自己 | 提示不能移除自己，拒绝移除 |
| 移除不存在的成员 | 视为不存在 |
| 普通成员查看成员列表 | 禁止访问 |
| 非小组成员操作 | 禁止访问 |
