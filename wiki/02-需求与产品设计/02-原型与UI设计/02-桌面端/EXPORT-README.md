## 四、覆盖范围（桌面端特有 + 核心业务）

桌面端基于 Web 管理后台复用同一套设计令牌与组件库，本原型重点呈现**桌面端增强能力**（原生窗口外壳 + 键盘效率 + 离线韧性），业务界面复用 Web 端：

**桌面端外壳与平台适配**
- 原生标题栏：`macOS` 交通灯按钮（hiddenInset，80px 避让区）/ `Windows` 自定义标题栏（40px，右上角窗口控制按钮）——右上角切换器可实时预览两平台差异
- 可折叠侧边栏（260px / 72px，复用 `--sidebar` 系列令牌，底部收起按钮 + `Cmd/Ctrl+B`）
- 全局命令面板（`Cmd/Ctrl+K`，支持页面跳转 / 主题切换 / 离线模拟）
- 原生右键菜单（表格行 `contextmenu` 呼出：查看详情 / 编辑 / 复制 ID / 删除）
- 离线提示横幅（`--state-warning`，40px，顶吸）、更新提示横幅（`--state-info`，48px）

**核心业务模块（复用 Web 端视图）**
1. 控制台 Dashboard — `#/dashboard`
2. 组织管理 Org — `#/org`（成员表格 + 右键菜单演示）
3. 团队管理 Team — `#/team`
4. 小组管理 Group — `#/group`
5. 角色权限 — `#/permissions`
6. 审计日志 Audit — `#/audit`（含未读 Badge）
7. 超级管理员 SuperAdmin — `#/superadmin`

**全局态**
- 认证态：右上角头像点击呼出登录浮层（`#login` 演示切换器）
- 主题态：明暗双主题（`⌘⇧T` / 标题栏按钮），通过 `[data-theme]` + CSS 变量重映射

## 五、设计对齐

- **设计语言**：Pinguo（克制、优雅、苹果风），单一品牌主色 `#007AFF`
- **令牌一致**：色彩 / 字体 / 间距 / 圆角 / 阴影 / 图标全部消费《00-全局设计规范》语义令牌，组件禁止硬编码原始色值
- **平台一致性**：macOS / Windows 差异遵循《02-桌面端设计规范》第 8 章（标题栏、菜单栏、侧边栏材质、字体回退、快捷键修饰键）
- **键盘友好**：`⌘K` 命令面板、`⌘B` 侧边栏、`⌘⇧T` 主题、`Esc` 关闭浮层，覆盖高频操作

## 六、质量结论（5 维自检）

| 维度 | 结论 | 说明 |
|------|------|------|
| 哲学 | PASS | 内容优先、克制用色，单一主色 + 中性灰阶 |
| 层次 | PASS | 标题栏 → 横幅 → 侧边栏 → 内容区层级清晰，留白建立层级 |
| 执行 | PASS | 令牌化、明暗双主题、Hover/按压态、圆角/阴影一致 |
| 特异性 | PASS | 桌面端特有：窗口外壳、命令面板、右键菜单、离线/更新横幅、平台切换 |
| 克制 | PASS | 无装饰性多色，状态色仅用于真实语义（离线/更新/角色标签） |
