## 二、文件信息

- **技术形态**：HTML + 内联 CSS + 内联 JS，零框架，原生 SPA（hash 路由）
- **设计基准**：iPhone 6 · 375px 屏幕宽 = 750rpx，原型 CSS 以 px 标注，严格按 `1px = 2rpx` 映射（圆角 16rpx→8px、间距 32rpx→16px、字号 32rpx→16px）

## 四、覆盖范围（小程序 · tabBar 4 + 微信生态页）

小程序定位为**轻量级微信入口**，依托微信能力（登录 / 订阅消息 / 分享 / 扫码），复杂管理引导至 Web / 桌面端。

**主框架**
- tabBar 导航（4 Tab，selectedColor `#007AFF` / color `#8E8E93`）：首页 / 组织 / 消息（未读 Badge）/ 我的
- 微信胶囊按钮示意（右上角）—— 对应真机 `navigationStyle` 自定义/默认适配

**页面**
1. 微信授权登录 `#/login` — 自定义导航、Logo + slogan、微信一键登录（绿色 `#07C160` 微信品牌色）、手机号绑定、用户协议勾选
2. 首页 `#/dashboard` — 当前组织卡、数据概览（成员/团队/待处理消息）、4 宫格快捷入口（扫码加入 / 邀请成员 / 消息 / 我的）、最近消息
3. 组织 `#/org` — 我的组织分组列表（头像 + 角色标签）+ 扫码加入入口
4. 组织详情 `#/org-detail` — 头部信息卡、数据统计、信息组、消息通知开关、退出组织、邀请/分享操作
5. 消息 `#/message` — 消息列表（未读圆点 / 数字 Badge）、全部已读
6. 消息详情 `#/message-detail` — 消息正文 + 操作按钮
7. 扫码加入 `#/scan` — 扫码框，调起 `wx.scanCode` 示意
8. 邀请分享 `#/invite` — 小程序码展示、分享给微信好友（`onShareAppMessage`）、保存小程序码
9. 我的 `#/profile` — 用户信息卡、当前组织切换、消息通知 / 深色模式开关、关于、退出登录

**全局态**
- 明暗双主题：我的页「深色模式」开关（`[data-theme]` 重映射，纯黑 OLED 背景）
- 底部 Action Sheet：组织详情「退出组织」、我的「当前组织」切换

## 五、设计对齐

- **设计语言**：Pinguo（苹果风），单一主色 `#007AFF`；仅微信登录按钮使用微信品牌绿 `#07C160`（平台约定，非装饰性用色）
- **小程序规范**：rpx 体系严格换算、tabBar 导航、Cell 分组列表（圆角 12pt / md、88rpx 按钮高 / 44px）、胶囊按钮避让
- **角色标签色板**：组织管理员蓝 `#007AFF`、团队管理员绿 `#34C759`、普通成员灰 `#AEAEB2` —— 与全局规范一致
- **状态标签**：活跃（绿）/ 待审核（橙）—— 状态色仅用于真实语义

## 六、质量结论（5 维自检）

| 维度 | 结论 |
|------|------|
| 哲学 | PASS · 内容优先、克制用色 |
| 层次 | PASS · 导航栏 → 卡片/列表 → tabBar 层级清晰 |
| 执行 | PASS · rpx→px 令牌化、圆角/阴影一致、明暗双主题 |
| 特异性 | PASS · 小程序特有：tabBar、微信授权登录、扫码加入、邀请分享、订阅消息/分享能力示意 |
| 克制 | PASS · 单一主色 + 中性灰阶，微信绿仅用于登录按钮 |

## 七、已知简化（演示边界）

- 微信登录、订阅消息、分享、扫码为**前端交互示意**，未调用真实微信 JS-SDK / 基础库；
- 小程序码为占位 SVG，非真实 `wxacode` 生成；
- 管理类操作按规范仅在 Web / 桌面端提供，小程序以查看与轻操作 + 引导为主；
- 下拉刷新、上拉加载为规范示意，未做完整实现。
