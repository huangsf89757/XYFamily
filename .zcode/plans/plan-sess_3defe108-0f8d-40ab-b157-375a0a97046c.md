### 03.02-数据库设计/数据库设计
- 基于 PRD 数据字典 15 张表，输出**完整 PostgreSQL DDL**（字段类型 BIGINT/VARCHAR/JSONB、约束、索引、外键）

### 03.03-中间件专项方案/中间件专项方案

- 对象存储（MinIO/S3）：头像文件上传
- 缓存策略：角色权限点缓存（42权限点热数据）

### 03.04-核心技术专项方案/核心技术专项方案
- **多租户隔离方案**（落地审查报告 P1-11）：X-Org-Id/X-Team-Id/X-Group-Id Header 上下文 + 应用层强制校验成员归属 + Token Claims 内嵌组织列表
- **RBAC 权限引擎**（落地 FR-PERM-002/003/004）：权限点校验中间件、数据范围控制（global/org/team/group 四级）、权限继承链（L5→L1）
- **JWT 鉴权链**（落地审查报告 P1-05）：Access Token 无状态短效(30min) + Refresh Token 存 sessions 表 + 黑名单机制
- **审计日志**（落地 FR-AUDIT）：27 种 operation_type、异步写入、append-only、1年保留+匿名化
- **账号注销任务**（落地审查报告 P1-08）：定时任务扫描到期账号、匿名化处理流程
